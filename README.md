# Дошка оголошень

Функціональний веб-застосунок дошки оголошень з пошуком, сортуванням, пагінацією, валідацією та видаленням.

## Установка та запуск

```bash
npm install
npx prisma generate
npx prisma migrate dev --name init
node app.js
```

Відкрийте http://localhost:3000

## Функціонал

- ✅ Список оголошень з пошуком, сортуванням, пагінацією (10/стор)
- ✅ Перегляд деталей + JS DELETE з confirm
- ✅ Форма створення з сервер/клієнт валідацією
- ✅ 404/500 обробка помилок
- ✅ Responsive дизайн

## Prisma Studio (перегляд БД)
```bash
npx prisma studio
```

## Тестування
1. Створіть оголошення (/announcements)
2. Перевірте пошук/сортування/пагінацію
3. Перегляньте деталі та видаліть
4. Спробуйте невалідні дані
